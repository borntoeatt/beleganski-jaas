# Contributing to beleganski-jaas

Thank you for your interest in contributing to the world's most important open source project.

## Ground Rules

1. All contributions must respect the **RAKIA PUBLIC LICENSE v1.0**
2. All PRs must be reviewed by [@alex-beleganski](https://github.com/alex-beleganski) — please allow 3-5 business days or until the water gets cold, whichever comes first
3. Do not remove the jacuzzi

## Getting Started

```bash
git clone https://github.com/borntoeatt/beleganski-jaas.git
cd beleganski-jaas
open index.html  # no build step needed, we are not animals
```

## Development Setup

Requirements:
- A browser
- Optionally: a jacuzzi
- Optionally: rakia

## Submitting a PR

1. Fork the repo (Alex won't, he's in the jacuzzi)
2. Create a branch: `git checkout -b feat/more-rakia`
3. Make your changes
4. Test from a jacuzzi if possible
5. Submit PR — fill out the template honestly

## Code Style

- Tabs vs spaces: doesn't matter, Alex won't review it anyway
- Max line length: unlimited, like Alex's rakia supply
- Comments: encouraged, especially if jacuzzi-related

## Reporting Bugs

Use the bug report template. First step is always: **check the jacuzzi.**

## Code of Conduct

Be excellent to each other. Drink responsibly. Stay hydrated (water, not just rakia).

## Questions?

Open an issue. Alex will get to it eventually.
